# 프론트엔드 과제

## 실행 방법

### 1. 패키지 설치

```sh
npm install
```

### 2. 개발 서버 실행

```sh
npm run dev
```

### 3. 빌드

```sh
npm run build
```

### 4. Lint 체크

```sh
npm run lint
```

## 📚 폴더 구조 Feature-Sliced Design

✅ **명확한 책임 분리** → 특정 역할에 따라 레이어를 나누어 코드의 가독성을 높임  
✅ **독립적인 기능 개발** → Feature 단위로 분리하여 팀 단위 개발 및 테스트 용이  
✅ **유지보수성 향상** → 코드 변경이 다른 부분에 영향을 주지 않도록 설계  
✅ **재사용성 증가** → 공통 모듈을 `shared/`에 정의하고 다양한 기능에서 활용
